# ZSH Install

# Suggested Programs

## NeoVim
1. sudo add-apt-repository ppa:neovim-ppa/stable
2. sudo apt-get update
3. sudo apt-get install neovim
4. sudo update-alternatives --install /usr/bin/vi vi /usr/bin/nvim 60
5. sudo update-alternatives --config vi
6. sudo update-alternatives --install /usr/bin/vim vim /usr/bin/nvim 60
7. sudo update-alternatives --config vim
8. sudo update-alternatives --install /usr/bin/editor editor /usr/bin/nvim 60
9. sudo update-alternatives --config editor
